지원 플랫폼
=======================

kea dhcp manager는 JAVA 17 이상의 거의 대부분의 플랫폼에서 지원합니다.

1. 실제로 테스트 된 플랫폼
    + debian 11
    + debian 12
    + ubuntu 20.04LTS
    + MAC OS 13
    + MAC OS 12

2. 실제로 테스트 되지 않은 플랫폼
    + arch linux
    + rocky linux
    + centos 7
    + etc ...

3. 지원 미정 플랫폼
    + windows 10, 11
